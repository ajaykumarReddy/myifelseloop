Latest vscode extensions for angular

Angular2-switcher is very helpfull to switch between the TS, HTML, CSS and Spec files very fast using the keyboard shotcuts.

  • alt+o(Windows) shift+alt+o(macOS) if on .ts|.css|.spec.ts: go to html

  • alt+i(Windows) shift+alt+i(macOS) if on .ts|.html|.spec.ts: go to css

  • alt+u(Windows) shift+alt+u(macOS) if on .css|.html|.spec.ts: go to ts

  • alt+p(Windows) shift+alt+p(macOS) if on .ts|.css|.html: go to spec.ts

Which will help you for autocompile properties and methods of ts file to HTMl.

json2ts is a VSCode Extension which converts a JSON to TypeScript interfaces.

Usage json2ts converts a JSON from clipboard to TypeScript interfaces with cmd+alt+V or ctrl+alt+V.

This extension is optimized for developers that use Angular and are no longer using AngularJS (Angular 1). If you type in AngularJS keywords, it will suggest an Angular solutions. This is by design and meant to help developers learn how AngularJS concepts and techniques map to Angular.
