C programming Keywords and Identifiers

In this tutorial you will learn keywords in C programming, Identifiers in C and also how to name a variable.

Keywords in C Programming

Keywords are also called as ‘Reserved Words’ whose meaning has already been explained to C compilers. Keywords cannot be used as a variable names or identifiers because if we do we are trying to give a new meaning to the keyword, which is not accepted by the compiler. In C language total 32 keywords are available. Below table shows all the 32 keywords.

auto break case char
const continue default do
double else enum extern
float for goto if
int long register return
short signed sizeof static
struct switch typedef union
unsigned void volatile while

We will discuss each of these keywords in later chapters where these keywords are relevant.

Indentifiers in C Language

Identifiers are names given to variables, structures and constants etc. Identifiers must be unique because they are created to given a unique name to a variable or constants or structures to identify it during the execution of a program.

Identifiers and keywords are different and you cannot use keyword as an identifier. You can give any name to identifiers except keywords. It is always a good practice to give a meaningful name to the identifiers because you and your team are easy to understand.

Examples:

Int currency;
Double mainBalance;

Here ‘currency’ and ‘mainBalance’ are two identifiers given to data types ‘int’ and ‘double’.

Rules for Creating Identifiers:

  1. An identifier must only contain alphabets, digits and underscores.
  2. First character of an identifier must be alphabet or underscore.
  3. In C language identifiers are case sensitive. For example, ‘money’ and ‘Money’ are two different identifiers.
  4. Identifiers must not be ‘keywords’.

Some of the example of valid Identifiers:

money

_money

account_balance

balance_123

Some of the example of invalid Identifiers:

123balance – must not start with number

int-char-double – must not be a keyword

balance money – must not contain a white space
