Migrating Angular Apps to RxJS v6

RxJs 5 Operators are not treeshackable for bundling So by migrating Rxjs5 to Rxjs6 you can make all Rxjs operators as treeshackable. So that bundle size will get decreased so will get more performance

I will be showing how to migrate from Rxjs5 to Rxjs6

Migration

Go to project directory and install rxjs-lint as below

yarn add rxjs-tslint

And then you need to do the following command in project directory.

rxjs-5-to-6-migrate -p src/tsconfig.app.json

Now all the rxjx5 code changed to rxjs6 so it can be treeshackable and using pipe operators its more cleaner way to implement all the operators.
