Steps to Create Angular Elements

Angular Elements are introduced in angular6 new features, using angular elements we can Create our own reuseble custom html elements which can be used in other application like React, Vue and other angular applications.

Now let start how to create angular elements

1. Step1: installation

create a new project using angular-cli as like below

ng new my-angular-elements

After project is created succesfully go to the project directory and run the following commands

npm install @angular/elements

To make it works in old browsers make some changes in polyfill as below

npm install @webcomponents/custom-elements
