Angular Http Operators for Caching

Create angular application using Angular-CLI as below commands

ng new my-http-operators

And then add HttpClientModule to imports array of root module as follows

After that import HttpClient service into your component and then make an http call

You can find all these code in live example at the end

Use of Http Operators

1. Import HttpClient.

2. Make the http get call to retrive all the github repos of angular.

3. Used rxjs map operators to transform results.

Install Http-Operators

yarn add http-operators

Lets talk about the operators

ShareAndCache

shareAndCache will be caching the repos data in localStorage once its gets from the server and share the stored data accross all the subscribers.

So, Once get the data and cache it and share to all others subscribption

KeepFresh

keepFresh will be getting the fresh data from server evvery 10ms and update the results.

RetryExponentialBackoff

retryExponentialBackoff will be used to stop making calls after 3 retry if there are any errors with service calls
