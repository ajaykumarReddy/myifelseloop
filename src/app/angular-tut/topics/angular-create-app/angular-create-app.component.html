Set up the Development Environment

We have great tools which will help us to create angular application quicker and faster. Before start creating application, make sure you install the below softwares.

Install Node.js and npm if they are not already on your machine.

To Verify the installation do the following

1) Open commend propt

2) node -v

3) npm -v

4) npm install -g @angular/cli

1. Create a new project

Open a terminal window and run the following command.

ng new my-app

Here my-app is the project name that you want to create.

2. Run the application

after successfully created project, go to the project folder in terminal and run the following commands

ng serve -o

Now you can see on the browser the working project
