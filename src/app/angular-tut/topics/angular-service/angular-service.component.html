Why Services

Services are a great way to share the data beteen components and widely used to get the data from server. components should not be used to get the data directly from the servers, Components are only for presentation logic..

Create Services

You can create service using the angular-cli as below

ng generate service httpService //httpService is service name here.

Injectable is a class level annotation and tells the angular that this class is a service and going to be injectable in other angular classes.

providedIn: ProvideIn Tells the angular that this service is registered at root level and not required to declare service in providers array.

When you provide the service at the root level, Angular creates a single, shared instance of ItemService and injects into any class that asks for it.

By doing this we will get loosely-coupled communication between classes
